# weed_detection_summer_2026
Basic Weed Detection Model 

## Problem Description: 
- Given images, the model must accurately determine where the weed is loacted. It must handle varying weed density, lighting conditions and growth stages of the weeds. Assume the Camera is positioned to it gives a full view of what is in front of the Robot (From the top view).

## Reccommended Progression:
  1. Determine the type of model (transfer learning or building it from scratch)
  2. Find a Reliable Weed detection dataset (We reccomend finding an additional small dataste you can overfit to for debugging purposes)
  3. Build the model and test on your dataset
